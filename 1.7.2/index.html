Platforms


WepSIM for Web works with the most popular modern Web Browsers:
Google Chrome 50+, Mozilla Firefox 45+, Microsoft Edge 30+, and Apple Safari 10+

An Android 5.0+ version version is also available at Google Play.

It source code is available at https://github.com/wepsim/wepsim.
