Plataformas


WepSIM funciona en los navegadores Web más populares: Google Chrome 50+, Mozilla Firefox 45+, Microsoft Edge 30+, and Apple Safari 10+

Una versión para Android 5.0+ está también disponible en la tienda de Google Play.

El código fuente también está disponible en https://github.com/wepsim/wepsim.

Tecnologías


WepSIM usa diferentes tecnologías como:

  • Frameworks: Jquery Mobile, Knockout, Bootstrap
  • Iconos: glyphicons, icon-icons
  • Plug-ins y extensiones: Bootbox, Editor CodeMirror, Spectrum ColorPicker, Bootstrap Notify, Biblioteca de visualización vis.js, Jquery Knob
